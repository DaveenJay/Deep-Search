Setup Instructions

To use Deep Search correctly:

  1. This application must be served through a web server due to CORS restrictions.
  2. You can use a simple local server like python -m http.server or Node's http-server.
  3. Alternatively, you can create a backend proxy server to handle API requests.
  4. Make sure to configure your API key in API.js
